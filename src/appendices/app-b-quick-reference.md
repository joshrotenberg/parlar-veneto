# Appendix B: Quick Reference

This appendix provides at-a-glance summaries of key Venetian grammar points. Keep it handy as you read, write, or speak.

## B.1 Pronunciation Essentials

| Symbol | Sound | Example |
|--------|-------|---------|
| **x** | /z/ (voiced s) | *xe* /ze/ (is) |
| **ł** | velarized l or /w/ | *beło* (beautiful) |
| **s** | always /s/ (unvoiced) | *casa* (house) |
| **z** | /ts/ or /dz/ | *mezo* (half) |
| **gn** | /ɲ/ (like Italian) | *gnente* (nothing) |
| **ś/sc** | /stʃ/ before e/i | *sciao* (bye) |

**Key differences from Italian:**

- No double consonants: *tuto* not *tutto*
- Final vowels often dropped: *pan* not *pane*
- Intervocalic v often drops: *laorar* not *lavorare*

## B.2 Articles

### Definite Articles

| | Masculine | Feminine |
|---|-----------|----------|
| Singular | *el* | *la* |
| Plural | *i* | *le* |

Before vowels: *l'* for both genders singular

### Indefinite Articles

| Masculine | Feminine |
|-----------|----------|
| *un* | *na* / *una* |

## B.3 Subject Pronouns and Clitics

| Person | Emphatic | Clitic | Required? |
|--------|----------|--------|-----------|
| 1sg | *mi* | - | No clitic |
| 2sg | *ti* | *te* | **Required** |
| 3sg m. | *lu/eło* | *el* | **Required** |
| 3sg f. | *ela/eła* | *la* | **Required** |
| 1pl | *nu/noaltri* | - | No clitic |
| 2pl | *vu/voaltri* | (*ve*) | Regional |
| 3pl m. | *lori/eli* | *i* | **Required** |
| 3pl f. | *lore/ełe* | *le* | **Required** |

**Key rule:** 2nd and 3rd person ALWAYS need the clitic before the verb.

- *Ti **te** parli.* (You speak.)
- *Lu **el** parla.* (He speaks.)

## B.6 Essere vs. Gavér: Quick Summary

| Use *GAVÉR* for | Use *ÈSSER* for |
|-----------------|-----------------|
| Possession: *Go na casa.* | Identity: *Son veneto.* |
| Age: *Go trent'ani.* | Location: *Son a casa.* |
| Physical states: *Go fame.* | Description: *Xe grando.* |
| Compound tenses (transitive) | Compound tenses (motion/change) |
| Reflexives (variation) | Reflexives (variation) |

## B.7 Negation

| Type | Pattern | Example |
|------|---------|---------|
| Simple | *no* + verb | *No vago.* |
| With clitic | *no* + clitic + verb | *No el va.* |
| Double | *no* + verb + neg. word | *No vedo gnente.* |
| Emphatic | *no* + verb + *miga* | *No go miga capìo.* |

**Negative words:**

- *gnente* (nothing)
- *nissun* (nobody)
- *mai* (never)
- *gnanca* (not even)

## B.8 Questions

### Yes/No Questions

Invert verb and clitic:

- Statement: *Ti te magni.* → Question: *Magni-to?*
- Statement: *El magna.* → Question: *Magna-lo?*

## B.9 Common Prepositions

| Venetian | Italian | English |
|----------|---------|---------|
| *de* | *di* | of, from |
| *a* | *a* | to, at |
| *da* | *da* | from, by |
| *in* | *in* | in (also "to" for locations) |
| *con* | *con* | with |
| *su* | *su* | on |
| *par* | *per* | for |
| *tra/fra* | *tra/fra* | between |
| *sensa* | *senza* | without |

**Note:** Venetian uses *in* where Italian uses *a/al*:

- *Vago in bar.* (I'm going to the bar.)

## B.13 Key Differences from Italian

| Feature | Italian | Venetian |
|---------|---------|----------|
| "There is" | *c'è* | *ghe xe* |
| Negation | *non* | *no* |
| "Also" | *anche* | *anca* |
| Subject clitics | Optional | Required (2nd/3rd) |
| "At the bar" | *al bar* | *in bar* |
| Double consonants | Yes | No |
| *gi-* sound | /dʒ/ | /z/ (*x*) |
