# Chapter 15: Imperative

The imperative (*imperativo*) is used for commands, requests, instructions, and exhortations. Venetian imperatives are common in everyday speech - from *varda!* (look!) to *tasi!* (shut up!) to *ndemo!* (let's go!).

## 15.1 Informal Imperative (Tu/Ti)

The informal singular imperative - used with people you address as *ti* - is the most common form.

### Formation

**-ar verbs:** Use the 2sg present indicative form (without the clitic):

| Infinitive | Imperative | English |
|------------|------------|---------|
| *magnar* | *Magna!* | Eat! |
| *parlar* | *Parla!* | Speak! |
| *vardar* | *Varda!* | Look! |
| *scoltar* | *Scolta!* | Listen! |
| *tasar* | *Tasi!* | Be quiet! |
| *spetar* | *Speta!* | Wait! |

**-er verbs:** Drop the infinitive ending:

| Infinitive | Imperative | English |
|------------|------------|---------|
| *véder* | *Vedi!* | See! / Look! |
| *scriver* | *Scrivi!* | Write! |
| *léxer* | *Lexi!* | Read! |
| *còrer* | *Cori!* | Run! |
| *creder* | *Credi!* | Believe! |

**-ir verbs:** Use the 2sg present form:

| Infinitive | Imperative | English |
|------------|------------|---------|
| *partir* | *Parti!* | Leave! |
| *vegnir* | *Vegni!* | Come! |
| *dormir* | *Dormi!* | Sleep! |
| *sentir* | *Senti!* | Hear! / Listen! |
| *finir* | *Finissi!* | Finish! |

### Common Irregular Imperatives

| Infinitive | Imperative | English |
|------------|------------|---------|
| *èsser* | *Sii!* | Be! |
| *gavér* | *Gabbi!* / *Abbi!* | Have! |
| *andar* | *Va!* / *Vatene!* | Go! / Go away! |
| *far* | *Fa!* / *Fà!* | Do! / Make! |
| *dir* | *Di!* / *Dì!* | Say! / Tell! |
| *dar* | *Da!* / *Dà!* | Give! |
| *star* | *Sta!* / *Stà!* | Stay! / Be! |
| *tòr* | *Tol!* / *Ciapa!* | Take! |
| *vegnir* | *Vegni!* / *Vien!* | Come! |

## 15.2 Formal Imperative (Lei/Vu)

The formal imperative uses subjunctive forms. In Venetian, the distinction between formal and informal is less rigid than in Italian, but formal forms exist.

### Singular Formal (with *vu* as polite singular)

Use the 3sg present subjunctive:

| Infinitive | Formal Imperative | English |
|------------|-------------------|---------|
| *magnar* | *Che 'l magna!* / *Magna!* | Eat! (formal) |
| *vegnir* | *Che 'l vegna!* | Come! (formal) |
| *scoltar* | *Scolte!* | Listen! (formal) |
| *dir* | *Diga!* | Tell! (formal) |

In practice, Venetians often use informal forms even in moderately formal situations, or soften commands with *par piaser* (please).

### Plural Formal/Informal

The 2pl imperative is used for groups:

| Infinitive | 2pl Imperative | English |
|------------|----------------|---------|
| *magnar* | *Magné!* | Eat! (you all) |
| *vardar* | *Vardé!* | Look! (you all) |
| *vegnir* | *Vegnì!* | Come! (you all) |
| *partir* | *Partì!* | Leave! (you all) |

## 15.3 First Person Plural (Let's...)

The "let's" form uses the 1pl present indicative, often with *ndemo* or *andemo*:

| Venetian | English |
|----------|---------|
| *Ndemo!* / *Andemo!* | Let's go! |
| *Magnemo!* | Let's eat! |
| *Bevemo!* | Let's drink! |
| *Parlemo!* | Let's talk! |
| *Vedemo!* | Let's see! |
| *Provemo!* | Let's try! |
| *Spetemo!* | Let's wait! |
| *Scomissiemo!* | Let's start! |

This is extremely common: *Ndemo* is one of the most frequent words in spoken Venetian.

## 15.4 Negative Imperative

### Informal Singular

Use *no* + infinitive (not the imperative form):

| Affirmative | Negative | English |
|-------------|----------|---------|
| *Magna!* | *No magnar!* | Don't eat! |
| *Parla!* | *No parlar!* | Don't speak! |
| *Va!* | *No andar!* | Don't go! |
| *Fa!* | *No far!* | Don't do! |
| *Di!* | *No dir!* | Don't say! |
| *Vegni!* | *No vegnir!* | Don't come! |

### Plural and Formal

Use *no* + the regular imperative form:

| Affirmative | Negative | English |
|-------------|----------|---------|
| *Magné!* | *No magné!* | Don't eat! (pl.) |
| *Ndemo!* | *No ndemo!* | Let's not go! |
| *Partì!* | *No partì!* | Don't leave! (pl.) |

## 15.5 Imperative with Pronouns

Object pronouns attach to the end of affirmative imperatives:

### Direct Object Pronouns

| Command | With Pronoun | English |
|---------|--------------|---------|
| *Varda!* | *Vardało!* | Look at it! (m.) |
| *Varda!* | *Vardała!* | Look at it! (f.) |
| *Ciapa!* | *Ciapało!* | Take it! |
| *Magna!* | *Magnało!* | Eat it! |
| *Scrivi!* | *Scrivime!* | Write me! |
| *Ciama!* | *Ciamame!* | Call me! |

### Indirect Object Pronouns

| Command | With Pronoun | English |
|---------|--------------|---------|
| *Di!* | *Dime!* | Tell me! |
| *Dà!* | *Dame!* | Give me! |
| *Porta!* | *Portaghe!* | Bring him/her! |
| *Scrivi!* | *Scrivighe!* | Write to him/her! |

### Combined Pronouns

| Command | With Pronouns | English |
|---------|---------------|---------|
| *Dà!* | *Damelo!* | Give it to me! |
| *Di!* | *Dimelo!* | Tell it to me! |
| *Porta!* | *Portaghelo!* | Bring it to him! |
| *Fa!* | *Famelo!* | Do it for me! |

### With Monosyllabic Imperatives

When *va*, *fa*, *da*, *di*, *sta* take pronouns, the consonant often doubles:

| Base | With Pronoun | English |
|------|--------------|---------|
| *Va!* | *Vatene!* / *Vane!* | Go away! |
| *Fa!* | *Fało!* / *Falo!* | Do it! |
| *Da!* | *Damelo!* | Give it to me! |
| *Di!* | *Dime!* | Tell me! |
| *Sta!* | *Stame a scoltar!* | Listen to me! |

### Negative Commands with Pronouns

In negative commands, pronouns can precede the infinitive or attach to it:

| Separated | Attached | English |
|-----------|----------|---------|
| *No me vardar!* | *No vardarme!* | Don't look at me! |
| *No lo far!* | *No farlo!* | Don't do it! |
| *No ghe dir!* | *No dirghe!* | Don't tell him! |

## 15.6 Common Imperative Expressions

### Everyday Commands

| Venetian | English | Context |
|----------|---------|---------|
| *Varda!* | Look! | Getting attention |
| *Scolta!* / *Senti!* | Listen! | Getting attention |
| *Tasi!* | Shut up! Be quiet! | Silencing |
| *Speta!* | Wait! | Holding |
| *Vien qua!* | Come here! | Summoning |
| *Va via!* / *Vatene!* | Go away! | Dismissing |
| *Ndemo!* | Let's go! | Departing |
| *Daghe!* | Go for it! Give it! | Encouraging |
| *Dai!* | Come on! | Urging |
| *Forsa!* | Come on! Strength! | Encouraging |

### Softened Commands

| Venetian | English |
|----------|---------|
| *Varda, par piaser...* | Look, please... |
| *Scolta un momento...* | Listen a moment... |
| *Fame un piaser...* | Do me a favor... |
| *Disime na roba...* | Tell me something... |
| *Provemo, dai!* | Let's try, come on! |

### Warnings and Prohibitions

| Venetian | English |
|----------|---------|
| *Ocio!* | Watch out! (lit. "eye") |
| *Atento!* / *Atenti!* | Careful! |
| *No tocar!* | Don't touch! |
| *No passar!* | Don't pass! / No entry! |
| *Ferma!* | Stop! |
| *Basta!* | Enough! Stop it! |
| *Basta cussì!* | That's enough! |

### Invitations and Offers

| Venetian | English |
|----------|---------|
| *Vegni dentro!* | Come in! |
| *Sentete!* | Sit down! (informal) |
| *Servite!* | Help yourself! |
| *Bevemo insieme!* | Let's drink together! |
| *Magna, magna!* | Eat, eat! |
| *Tol!* / *Ciapa!* | Take! Here! |

## 15.7 Imperative in Exclamations

The subjunctive/imperative appears in exclamatory wishes:

| Venetian | English |
|----------|---------|
| *Che Dio te benedisa!* | God bless you! |
| *Che 'l vaga al diàvoło!* | To hell with him! |
| *Viva Venesia!* | Long live Venice! |
| *Che la vaga ben!* | May it go well! |

## Summary

### Regular Imperative Forms

| | -ar (*magnar*) | -er (*véder*) | -ir (*partir*) |
|---|----------------|---------------|----------------|
| 2sg (ti) | *Magna!* | *Vedi!* | *Parti!* |
| 2pl (vu) | *Magné!* | *Vedé!* | *Partì!* |
| 1pl (nu) | *Magnemo!* | *Vedemo!* | *Partimo!* |

### Common Irregular Imperatives 2


| Infinitive | 2sg | 1pl |
|------------|-----|-----|
| *andar* | *Va!* | *Ndemo!* |
| *far* | *Fa!* | *Femo!* |
| *dir* | *Di!* | *Disemo!* |
| *dar* | *Da!* | *Demo!* |
| *vegnir* | *Vegni!* / *Vien!* | *Vegnimo!* |
| *star* | *Sta!* | *Stemo!* |

### Negative Imperative

| Person | Formation | Example |
|--------|-----------|---------|
| 2sg | *No* + infinitive | *No magnar!* |
| 2pl | *No* + imperative | *No magné!* |
| 1pl | *No* + imperative | *No magnemo!* |

### Pronoun Attachment

- Affirmative: pronouns attach to verb end (*Vardało!*, *Dime!*)
- Negative: pronouns precede infinitive or attach (*No me vardar!* / *No vardarme!*)
